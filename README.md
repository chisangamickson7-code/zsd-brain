# ZSD-Brain 🎤
Zambia Singing Dataset Brain - AI Music Assistant

This project trains an AI to sing using your own voice.

## Dataset Structure
`my_singing_dataset/` ← THIS IS THE MAIN FOLDER
│
├── wavs/           ← FOLDER 1: Put all .wav audio files here
│   └── segment_001.wav
│
├── transcripts/    ← FOLDER 2: Put all .txt lyric files here  
│   └── segment_001.txt
│
└── midis/          ← FOLDER 3: Put all .mid note files here
    └── segment_001.mid

## How to Run
1. `pip install -r requirements.txt`
2. `python app.py`
